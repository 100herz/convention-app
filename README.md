## API calls we use

### Categories

```plain
/wp-json/wp/v2/categories?per_page=99
```

### Posts

```plain
/wp-json/wp/v2/posts?per_page=10
```

### Posts - ACF

```plain
/wp-json/acf/v3/posts?per_page=10
```
